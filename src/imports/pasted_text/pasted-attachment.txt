Project Title

Rosario Dairy Management System

Design a complete UI/UX prototype for a web-based Integrated Inventory, Point-of-Sale, Order Management, and Machine Learning-Based Predictive Analytics System for Rosario Dairy.

The system is intended for a dairy business that manages perishable products using FEFO (First Expired, First Out) inventory management and SARIMA-based predictive analytics.

Create two separate user interfaces:

Administrator Dashboard
Staff Dashboard

Both dashboards should share the same branding and design language while displaying different modules according to user roles.
